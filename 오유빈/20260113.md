# React vs Vue  
## 개념과 차이점 정리
## 1. 공통 개념

React와 Vue는 모두 프론트엔드 JavaScript 기술로, 동적인 웹 UI를 만들기 위한 도구

- 컴포넌트 기반 개발
- 상태에 따른 화면 자동 갱신
- Virtual DOM 사용
- SPA(Single Page Application) 구현

## 2. React 개념

React는 UI(View)를 만들기 위한 라이브러리로 화면 렌더링에 집중하며 나머지 기능은 개발자가 선택할 수 있음

### React 핵심 특징
- Meta(Facebook)에서 개발
- JSX 문법 사용 (JavaScript 안에 HTML 작성)
- 상태(State) 중심 렌더링
- 높은 자유도와 확장성

### React 예시
```jsx
function Hello() {
  const [count, setCount] = useState(0)

  return (
    <button onClick={() => setCount(count + 1)}>
      클릭 {count}
    </button>
  )
}
```

## 3. Vue 개념

Vue는 UI 개발을 위한 프레임워크로 화면 구성 방식과 구조를 공식적으로 제공

### Vue 핵심 특징
- Evan You가 개발
- HTML 기반 템플릿 문법
- 직관적인 데이터 바인딩
- 낮은 진입 장벽

### Vue 예시
```vue
<template>
  <button @click="count++">
    클릭 {{ count }}
  </button>
</template>

<script setup>
import { ref } from 'vue'
const count = ref(0)
</script>
```

## 4.핵심 차이점 비교
| 구분       | React       | Vue                 |
| -------- | ----------- | ------------------- |
| 성격       | 라이브러리       | 프레임워크               |
| 문법       | JSX (JS 중심) | Template (HTML 중심)  |
| 자유도      | 높음          | 비교적 낮음              |
| 진입 장벽    | 상대적으로 높음    | 낮음                  |
| 공식 도구 제공 | 거의 없음       | Router, Store 공식 제공 |
| 학습 난이도   | 가파름         | 완만                  |
