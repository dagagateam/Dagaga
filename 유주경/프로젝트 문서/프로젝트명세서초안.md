# 프로젝트 명세서 초안

---

## 1. 프로젝트 개요

- **서비스 정의:** 다문화 가정 학부모의 한국어 발화 자신감 향상을 위한 생성형 AI 기반 쉐도잉 및 커뮤니티 플랫폼
- **핵심 가치:** "완벽하지 않아도 괜찮아, 입 밖으로 소리 내어 말하는 연습"
- **타겟 유저:**
    - 한국어 소통에 두려움을 느끼는 다문화 가정의 부모 (주로 취학 전/후 자녀를 둔 부모)
    - 지역 별 육아 및 다문화 혜택 정보를 얻고자 하는 학부모

## 2. 사용자 페르소나 (예시)

- **이름:** 레티 흐엉 (32세, 베트남 출신)
- **상황:** 초등학교 입학을 앞둔 자녀가 있음. 학부모 상담이나 병원 방문 시 하고 싶은 말이 머릿속에는 모국어로 맴드는데, 한국어로 번역해서 말하려니 문법이 틀릴까 봐 입이 떨어지지 않음.
- **니즈:** 정확한 문법 교정보다는, 당장 내일 선생님께 할 말을 미리 연습해서 '자신 있게' 말하고 싶음.

---

## 3. 주요 기능 명세 (Functional Requirements)

### A. 학습 기능 (Core Feature)

**목표:** 개인화된 스크립트 제공 및 단계별 쉐도잉을 통한 발화 유도

### 1. 상황별 예시 따라 읽기 (LLM 기반 시나리오 학습)

- **기능 설명:** 사용자가 선택한 카테고리(학교, 병원, 관공서, 마트 등)와 세부 상황에 맞춰 LLM이 적절한 대화문을 생성합니다.
- **세부 프로세스:**
    1. **상황 선택:** 카테고리 선택 (예: 학교) → 상황 선택 ( 예: 자녀의 준비물 물어보기)
    2. **스크립트 생성:** LLM이 해당 상황에 맞는 자연스러운 한국어 문장 3~5개 생성
    3. **단계별 학습 (3-Step):**
        - *Step 1 (듣기):* TTS로 자연스러운 한국어 발음 듣기
        - *Step 2 (끊어 읽기):* 의미 단위(Chunk)로 쪼개진 문장을 따라 읽기 (UI상 하이라이팅 제공)
        - *Step 3 (전체 읽기):* 전체 문장을 이어서 말하기
    4. **피드백:** 발음 정확도 점수보다는 **'완주 여부', '목소리 크기', '리듬감'** 등을 시각화하여 칭찬 위주의 피드백 제공 (동기 부여)

### 2. "하고 싶은 말" 맞춤 변환 (Native to Korean)

- **기능 설명:** 기존 학습 앱의 정해진 예문이 아니라, 사용자가 실제 상황에서 하고 싶은 말을 모국어로 입력하면 한국어 스크립트로 변환하여 학습합니다.
- **세부 프로세스:**
    1. **입력:** 사용자가 자신의 모국어(예: 베트남어, 중국어 등)로 음성 녹음 또는 텍스트 입력 (예: "우리 아이가 어제부터 배가 아파서 학교에 못 가요.")
    2. **번역 및 생성 (LLM):** * 입력된 내용을 한국어 문맥과 존댓말/반말 상황에 맞춰 적절한 문장으로 변환.
        - 여러 버전 제공 (예: 정중한 버전, 친근한 버전).
    3. **개인화 학습:** 생성된 문장을 즉시 '따라 읽기' 모드로 전환하여 연습.
    4. **보관:** '내 단어장'처럼 '내 문장'에 저장하여 반복 학습.

### B. 커뮤니티 기능 (Community Feature)

**목표:** 정보 비대칭 해소 및 심리적 유대감 형성

### 1. 지역 기반 정보 공유 (Local Info)

- **기능 설명:** 사용자의 거주 지역(GPS 또는 설정 기반)에 맞는 다문화 혜택 및 육아 정보를 공유합니다.
- **세부 내용:**
    - **공공 데이터 연동:** 지자체별 다문화 가족 지원 센터의 공지사항 크롤링/연동.
    - **사용자 게시판:** "00동 주민센터에서 한국어 교실 열렸대요", "00소아과 선생님이 친절해요" 등 생활 밀착형 정보 공유.
    - **자동 번역:** 게시글을 사용자의 주 언어로 볼 수 있는 번역 버튼 제공.

### 2. 스크립트 공유 SNS (Script Sharing)

- **기능 설명:** 자신이 학습 기능(A-2)을 통해 만든 '나만의 문장'과 실제 사용 후기를 공유합니다.
- **세부 내용:**
    - **공유 예시:**
        - *상황:* "학부모 총회 때 자기소개"
        - *내가 만든 스크립트:* "안녕하세요, 지민이 엄마입니다. 한국말이 서툴지만 열심히 듣겠습니다."
        - *후기:* "이렇게 말했더니 다른 엄마들이 웃으며 반겨줬어요!"
    - **스크랩:** 다른 사용자가 올린 좋은 표현을 내 학습 목록으로 가져오기(Fork).
