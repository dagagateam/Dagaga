# AI
## AI를 얼마나 잘 깎아야 할까?
수상이 얼마나 ai를 잘 깎는지에 따라 갈린다 봐도 무방하다고 하셨는데 그러면 어떤 모델을 어떻게 튜닝하고 성능을 뽑고 어디까지 설정할지 정해야 한다

회의에서 질문 특화와 답변 특화 중에 답변 특화 중 결국 답변을 하면서 자신감을 얻는 것을 생각해서 답변 특화로 만들기로 함

1. 선생님 상담에서 나오는 보편적인 질문, 내용들을 모아서 docs로 만듦
2. RAG로 해당 docs에서 먼저 검색하고 답변 쉐도우 텍스트를 생성
  - 여기서 출력은 짱구는 ~해요 + 하단의 작은 글씨로 모국어 출력
  - 또한 외국인은 어디까지 끊어서 읽을 수 있는지 (조사/단어 어디까지 가능한지) 조사

3. +@

## 상황
한국어로 말하고 싶은 단어/문장을 모국어(ex. 베트남어, 중국어 등)로 입력
해당 내용을 상황에 맞게 번역한 한국어 + 모국어가 매칭되어 생성됨
사용자는 한국어를 잘 모르다 보니 어디 부분에서 어떻게 해석 되는지 모를 수 있음
- 추가로 어떤 기준으로 문장을 끊어서 매칭해야 할지 더 알아봐야 함

## ai 현황과 문제
whisper 모델이 다국어 지원모델이다보니 '나는 사과가 먹고 싶어요'를 'Na는 사과가 먹고 싶어요'처럼 다른 언어로 번역하거나 틀리게 출력되는 경우가 종종 있음
-> 위 상황을 보정하는 알고리즘을 짤 것인지, 아니면 출력에 후보정을 할 것 인지 또 다른 방법을 적용할 것 인지 결정해야 하며 빠른 시일 내 최신화된 모델들이 어느정도의 인식 성능을 가지고 있는지 모델 확인을 진행해야 함


## 인터뷰 내용 반영
한국어를 배울 때 쓰기가 가장 어려웠다, 한국어를 조금만 할 수 있어도 일을 하다보니 여러가지 배움터나 상황이 있더라도 시간을 내기가 어렵다
언어의 배움이 쉬운 영역이 아님 + 시간을 많이 투자할 수 없음 으로 인해 오랜 시간이 지나도 배우려고 노력하지 않는다면 성장이 더딜 수 밖에 없음

온라인으로 처천히 알려주는 건 더 좋다 + 다른 일을 병행하거나 동시에 할 수 없는 상황이 있기 때문에 기억에 어려움이 있을 수 있다
