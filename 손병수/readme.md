- 잘 되는 부분
  - 매우 틀린 발음, 다른 단어로 들릴 정도의 오류, 완전히 누락된 음절 등은 잘 탐지해서 피드백을 줄 수 있음
  - 어떤 단어, 문장에서 인식 실패가 반복되는지 대략적으로 파악 가능


- 어려운 부분
  - 망설임, 매우 느린 말하기, 문장이 끊기는 패턴이 일정하지 않을 경우
  - 단순히 구별은 할 수 있으나 질적인 평가가 애매할 수 있음

워낙 기술의 발전이 빠르다 보니 최근에 나오는 모델들, 크기, 종류 등에 따라 비교를 해보면서 어느 정도의 성능을 가지고 있고 상황에 따라 노이즈를 추가하여 성능을 낮추거나 기존의 있던 모델들에 서비스에 필요한 내용들만 추가학습을 시키는 방식으로 진행을 할 예정
