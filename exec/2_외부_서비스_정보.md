# 외부 서비스 정보

## 1. Google OAuth 2.0 (소셜 로그인)

사용자의 간편 로그인 및 회원가입을 위해 사용합니다.

*   **서비스 제공자**: Google Cloud Platform (GCP)
*   **설정 페이지**: [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
*   **필요 권한 (Scopes)**:
    *   `.../auth/userinfo.email`
    *   `.../auth/userinfo.profile`
*   **설정 방법**:
    1.  프로젝트 생성 및 'OAuth 동의 화면' 설정
    2.  '사용자 인증 정보' > 'OAuth 클라이언트 ID' 생성 > '웹 애플리케이션' 선택
    3.  **승인된 리디렉션 URI** 설정 (필수): `https://i14b110.p.ssafy.io/login/oauth2/code/google`
    4.  발급된 `Client ID`와 `Client Secret`을 `.env` 파일에 등록

---

## 2. Google Gemini API - GMS

다국어 채팅 번역 및 텍스트 분석을 위해 사용합니다.

*   **모델**: `gemini-2.0-flash-lite`
*   **설정 방법**:
    *   발급된 GMS API 키를 `GEMINI_API_KEY` 환경 변수에 등록
*   **주의사항**:
    *   분당 요청 횟수 제한이 있을 수 있습니다.

---

## 3. Gmail SMTP

비밀번호 찾기, 본인 인증 이메일 발송을 위해 사용합니다.

*   **서비스 제공자**: Google Gmail
*   **설정 방법**:
    1.  Google 계정 관리 > 보안 > '2단계 인증' 켜기
    2.  검색창에 '앱 비밀번호' 검색
    3.  앱 이름 설정(예: `Dagaga`) 후 생성
    4.  생성된 16자리 비밀번호를 `MAIL_PASSWORD` 환경 변수에 등록
*   **주요 포트**: `587` (STARTTLS)
