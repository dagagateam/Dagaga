# 외부 서비스 정보

## 1. Google OAuth 2.0 (소셜 로그인)

사용자의 간편 로그인 및 회원가입을 위해 사용합니다.

*   **서비스 제공자**: Google Cloud Platform (GCP)
*   **설정 페이지**: [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
*   **필요 권한 (Scopes)**:
    *   `.../auth/userinfo.email`
    *   `.../auth/userinfo.profile`
*   **설정 방법**:
    1.  프로젝트 생성 및 'OAuth 동의 화면' 설정
    2.  '사용자 인증 정보' > 'OAuth 클라이언트 ID' 생성 > '웹 애플리케이션' 선택
    3.  **승인된 리디렉션 URI** 설정 (필수): `https://i14b110.p.ssafy.io/login/oauth2/code/google`
    4.  발급된 `Client ID`와 `Client Secret`을 `.env` 파일에 등록

---

## 2. Google Gemini API - GMS

다국어 채팅 번역 및 텍스트 분석을 위해 사용합니다.

*   **모델**: `gemini-2.0-flash-lite`
*   **설정 방법**:
    *   발급된 GMS API 키를 `GEMINI_API_KEY` 환경 변수에 등록
*   **주의사항**:
    *   분당 요청 횟수 제한이 있을 수 있습니다.

---

## 3. Gmail SMTP

비밀번호 찾기, 본인 인증 이메일 발송을 위해 사용합니다.

*   **서비스 제공자**: Google Gmail
*   **설정 방법**:
    1.  Google 계정 관리 > 보안 > '2단계 인증' 켜기
    2.  검색창에 '앱 비밀번호' 검색
    3.  앱 이름 설정(예: `Dagaga`) 후 생성
    4.  생성된 16자리 비밀번호를 `MAIL_PASSWORD` 환경 변수에 등록
*   **주요 포트**: `587` (STARTTLS)

---

## 4. 자체 구축 AI 서비스

본 프로젝트는 두 개의 AI 서버를 자체적으로 구축하여 운영합니다.

### 4-1. ASR Server (음성 인식)

**역할**: 음성 메시지를 텍스트로 변환 (Speech-to-Text) 및 발음 평가

*   **서버 파일**: `ai/server/unified_asr.py`
*   **Docker 이미지**: NVIDIA CUDA 12.4.1 + cuDNN (Ubuntu 22.04)
*   **포트**: `4000` (컨테이너 내부) → `8000` (호스트)
*   **엔드포인트**:
    *   `POST /transcribe`: 음성 파일을 텍스트로 변환
    *   `POST /evaluate`: 발음 평가 (정확도, 유창성, 전체 점수)
    *   `GET /health`: 서버 상태 확인
*   **사용 모델**:
    *   **Base Model**: `openai/whisper-large-v3`
    *   **Finetuned Model**: LoRA 적용된 한국어 특화 모델 (`/app/finetuned`)
*   **주요 기술 스택**:
    *   PyTorch 2.5 (CUDA 12.4)
    *   Transformers (Hugging Face)
    *   FastAPI + Uvicorn
*   **환경 변수**:
    *   `.env` 파일 필요 (API 키, 모델 경로 등)

### 4-2. RAG GMS Server (발음 가이드)

**역할**: RAG(Retrieval-Augmented Generation) 기반 한국어 발음 가이드 제공

*   **서버 파일**: `ai/server/rag_gms.py`
*   **Docker 이미지**: Python 3.10 Slim
*   **포트**: `3000` (컨테이너 내부) → `7000` (호스트)
*   **엔드포인트**:
    *   `POST /tokenize`: 한국어 텍스트를 형태소 단위로 분리
    *   `POST /pronunciation-guide`: RAG 기반 발음 가이드 생성
    *   `GET /health`: 서버 상태 확인
*   **사용 모델**:
    *   **LLM**: Google Gemini 2.0 Flash Lite (API)
    *   **Embeddings**: ChromaDB 벡터 스토어
*   **RAG 데이터소스**:
    *   `Korean_guide.pdf`: 한국어 발음 특성 가이드
    *   `chroma_db/`: 사전 구축된 벡터 데이터베이스
*   **주요 기술 스택**:
    *   LangChain
    *   ChromaDB
    *   FastAPI + Uvicorn
*   **환경 변수**:
    *   `.env` 파일에 `GEMINI_API_KEY` 필수

*   **호스트**: `jukang.duckdns.org`
*   **ASR 접속 URL**: `http://jukang.duckdns.org:8000`
